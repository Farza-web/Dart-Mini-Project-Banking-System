# Dart-Mini-Project-Banking-System
A simple console-based Banking System built with Dart. The application allows users to create accounts, deposit and withdraw money, check balances, and view account information. This project is designed to practice fundamental Dart programming concepts such as Maps, loops, functions, conditional statements, and user input handling.
## Features
- Create bank account
- Deposit money
- Withdraw money
- Check account balance
- View all accounts

## Concepts Used
- Map
- Functions
- Loops
- Conditional statements
- User input

## How to Run

1. Install Dart SDK
2. Run the program:
Banking_System_Main.dart
